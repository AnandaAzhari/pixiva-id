# QA Checklist

## Purpose

Manual verification steps used before each release.

## Pre-Release Checks

- [ ] Welcome screen renders correctly.
- [ ] Camera permission flow works in supported browsers.
- [ ] Countdown triggers capture.
- [ ] Captured image is displayed on the review screen.
- [ ] Download saves a valid image file.
- [ ] Print opens a print dialog with the captured image.
- [ ] Share shows the placeholder notice.
- [ ] Done resets the session back to Welcome.

## Device Checks

- [ ] iPad Mini 6 (Safari).
- [ ] Chrome on desktop.
- [ ] Edge on desktop.
