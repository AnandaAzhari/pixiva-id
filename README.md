# PIXIVA.ID

MamQi Booth is a web-based photobooth application for promoting Risol MamQi. It is designed to run on an iPad Mini 6 today and evolve into a commercial rental photobooth platform.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Vercel

## Current Status

Sprint 1 preparation is complete. The project has a clean feature-based structure and is ready for the first feature implementation.

## Roadmap

- Build the core camera capture experience.
- Add countdown and frame selection.
- Support photo downloads and QR sharing.
- Add GIF and boomerang capture modes.
- Prepare rental, multi-event, and multi-client capabilities.
